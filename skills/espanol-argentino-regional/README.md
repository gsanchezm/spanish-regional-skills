# español-argentino-regional

Skill que permite a Claude responder en las variedades regionales del **español argentino**, de forma auténtica y respetuosa, para asistencia real o entretenimiento.

## Regiones incluidas

- **Rioplatense / porteño** — Buenos Aires, Rosario, La Plata (default)
- **Cordobés** — Córdoba (la tonada del alargamiento pretónico)
- **Cuyano** — Mendoza, San Juan
- **NOA / noroeste** — Salta, Jujuy, Tucumán (sustrato quechua)
- **Litoral / guaranítico** — Corrientes, Misiones, Chaco (sustrato guaraní)

Más una ficha de **rasgos generales** (base común y registro "argentino neutro").

## Lo esencial: el voseo

El voseo argentino es **pronominal + verbal** y es la forma culta normal: *vos tenés, vos sos, vení, dale*. Nunca *tú tienes*, y ojo: no es el voseo chileno (*tú tenís*). La skill insiste en esto porque es el error de autenticidad más común.

## Principios

- **Inteligibilidad primero**: glosa lo que pueda no entenderse.
- **Tendencias, no reglas**: nadie usa todos los rasgos en cada frase.
- **Respeto**: nunca ligar un acento a ignorancia; honrar el sustrato quechua (NOA) y guaraní (litoral) como vivo; la tonada cordobesa no implica baja educación.
