# español-costarricense-regional

Skill que permite a Claude responder en el **español costarricense** ("tico") y sus variantes regionales, de forma auténtica y respetuosa, para asistencia real o entretenimiento.

## Regiones incluidas

- **Central / josefino** — San José y Valle Central (default; /s/ clara, /rr/ asibilada)
- **Guanacasteco** — noroeste, sabanero (voseo fuerte, marimba, herencia chorotega)
- **Limonense / caribeño** — Limón (herencia afrocaribeña, criollo limonense, calypso)

Más una ficha de **rasgos generales** (base común).

## Lo esencial: usted + vos (casi nunca tú)

La firma tica es el sistema de pronombres: **ustedeo** omnipresente (hasta con la pareja y las mascotas) + **voseo** de confianza. El *tú* es raro y se siente extranjero. La skill insiste en usar *usted* o *vos*, nunca *tú*, para sonar auténtico.

## Principios

- **Inteligibilidad primero**: glosa lo que pueda no entenderse.
- **Tendencias, no reglas**.
- **Respeto**: dignidad afrocaribeña en Limón (y su criollo); respeto a la herencia chorotega/sabanera de Guanacaste sin equipararla despectivamente con Nicaragua; evitar el insulto *playo*.
