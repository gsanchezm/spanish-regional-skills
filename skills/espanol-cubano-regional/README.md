# español-cubano-regional

Skill que permite a Claude responder en el **español cubano** y sus variantes regionales, de forma auténtica y respetuosa, para asistencia real o entretenimiento.

## Regiones incluidas

- **Occidental / habanero** — La Habana y occidente (default; geminación de /r/, "¿qué bolá?")
- **Oriental / santiaguero** — Santiago, Holguín, Guantánamo (lambdacismo /r/→/l/, entonación cantada)

Más una ficha de **rasgos generales** (base común).

## Lo esencial: tuteo + fonética caribeña

El cubano usa **tuteo, nunca voseo** (*tú tienes*), con el rasgo estrella del **pronombre redundante** (*¿qué tú quieres?*). La fonética caribeña es fuerte: **caída de /s/** (*¿cómo etá?*), **geminación de /r/** en occidente (*puetta*) vs. **/r/→/l/** en oriente (*amol*). La skill insiste en esto.

## Principios

- **Inteligibilidad primero**: el cubano aspira mucho; glosa generosamente.
- **Tendencias, no reglas**.
- **Respeto**: honrar la **raíz afrocubana** (*asere* viene del abakuá; el son y la rumba son afrocubanos); evitar la **caricatura política**; no tratar el habla oriental como "menos correcta".
