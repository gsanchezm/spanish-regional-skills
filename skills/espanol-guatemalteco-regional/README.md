# español-guatemalteco-regional

Skill que permite a Claude responder en el **español guatemalteco** ("chapín"), de forma auténtica y respetuosa, para asistencia real o entretenimiento.

## Regiones incluidas

- **Central / chapín** — Ciudad de Guatemala y estándar nacional (default; /s/ clara, voseo, "un mi…")
- **Oriente** — Jutiapa, Jalapa, Chiquimula (ladino-ganadero, algo de aspiración)
- **Occidente / altiplano** — Xela, Huehuetenango, Quiché (altiplano maya)

Más una ficha de **rasgos generales** (base común).

## Lo esencial: voseo + /s/ retenida + el "un mi…"

Tres rasgos marcan al chapín: el **voseo** (*vos tenés, vení*); la **/s/ clara y retenida** (NO se aspira, a diferencia de El Salvador/Honduras — esto es lo que más lo distingue en Centroamérica); y la construcción de **sustrato maya "un mi amigo", "una mi tacita"** (artículo + posesivo). La skill insiste en esto.

## Principios

- **Inteligibilidad primero**: glosa el "un mi…" y los mayismos.
- **Tendencias, no reglas**.
- **Respeto**: dignidad maya (22+ lenguas vivas; el español con influencia maya NO es "roto"); prohibido *indio* como insulto; *cerote/shuco* cambian con el tono (fuera de asistencia); no reproducir la división ladino/indígena.
