# español-ecuatoriano-regional

Skill que permite a Claude responder en las variedades regionales del **español ecuatoriano**, de forma auténtica y respetuosa, para asistencia real o entretenimiento.

## Regiones incluidas

- **Andino / serrano** — Quito, Sierra, Cuenca, Loja (default; sustrato Kichwa, /rr/ asibilada)
- **Costeño / guayaco** — Guayaquil y la Costa (/s/ aspirada, ritmo rápido)

Más una ficha de **rasgos generales** (base común, con nota amazónica).

## Lo esencial: pronombres + el gerundio andino

La Sierra usa *tú/usted* con **voseo pronominal** (*vos tienes*, no el *vos tenés* argentino) y **leísmo**; la Costa usa *tú/ustedes*. El rasgo más característico de la Sierra es el **gerundio andino** del sustrato Kichwa: *"dame trayendo"*, *"dame abriendo"*. La skill insiste en esto.

## Principios

- **Inteligibilidad primero**: glosa el gerundio andino y lo que pueda no entenderse.
- **Tendencias, no reglas**.
- **Respeto**: honrar el Kichwa (no es "español roto"); nada de *longo/indio*; no usar la rivalidad serrano/mono como burla; dignidad afroecuatoriana (Esmeraldas) y montuvia.
