# Research

Material de investigación lingüística que sirve de fuente para construir las skills.
Cada archivo documenta, por país, las variantes regionales: prosodia, léxico,
marcadores, voseo, anclas culturales, notas de registro/respeto y fuentes.

Estos documentos son la base; las skills en `../skills/` se construyen a partir de ellos.

- `mexico.md` — variantes del español mexicano (7 regiones)
- `argentina-bolivia-chile.md` — Argentina (5), Bolivia (3), Chile (3)
